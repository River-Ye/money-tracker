# README

* Ruby version 2.6.3

* Rails version 6.0.1

* Database initialization psql (PostgreSQL) 12.1

# How to use

1. clone project
```bash
git clone https://github.com/River-Ye/money-tracker.git
```

2. `rails bundle`

3. `yarn install --check-files`

4. `rails db:migrate:reset`

5. run project
```bash
foreman start

# or

foreman s
```
